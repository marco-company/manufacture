This module enforces that each Manufacturing Order (MO) has a unique lot number
for its finished products.

A lot generated by one MO cannot be reused in another MO, and this constraint
is applied when validating the production.

Additionally, to improve user usability, the domain of the lot_producing_id
field has been updated to ensure only lot numbers that have not been used
are selectable, only in case the product is tracked by lots.
